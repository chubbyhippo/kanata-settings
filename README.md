# kanata-settings

[kanata](https://github.com/jtroo/kanata) configs with "timerless" home row mods
(urob-style: mods only arm after a 250 ms typing pause, so fast typing never
misfires or reorders letters), plus NAV / SYM / NUM / FUN layers, neighbour-key
combos, a mouse layer, Num Word, and a plain mode for lending the keyboard.

Two ports of the same layout — pick your OS:

| Folder | Config | Docs |
|---|---|---|
| [`wins/`](wins/) | `wins.kbd` — Windows (GASC home row: Win Alt Shift Ctrl) | [wins/README.md](wins/README.md) |
| [`mac/`](mac/) | `mac.kbd` — macOS (GASC home row: Cmd Opt Shift Ctrl, Karabiner driver + LaunchDaemons installer) | [mac/README.md](mac/README.md) |

The two configs mirror each other structurally; each README documents its
platform's install steps, layer maps, combos, and troubleshooting.
